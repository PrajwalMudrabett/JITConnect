# JITConnect React

A social networking platform for Jyothy Institute of Technology built with React, featuring Ferrari-inspired premium UI design.

## Features

- Role-based authentication (Student/Faculty/Alumni/Department)
- Post creation and sharing
- Real-time messaging
- Connection management
- User profiles
- Search and explore
- JIT campus backgrounds and branding

## Tech Stack

- React 19
- React Router DOM 7
- Vite
- LocalStorage for data persistence

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Open your browser and navigate to the URL shown in terminal (usually http://localhost:5173)

## Build for Production

```bash
npm run build
```
